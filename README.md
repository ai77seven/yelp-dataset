# yelp-dataset
# 安装Nodejs
地址(https://nodejs.org/en/)
Mac
```
brew install node
```
Start the project web


```
# install vue-cli
$ npm install --global vue-cli
# install dependencies and go!
$ git clone git@github.com:visgroup10/yelp-dataset.git
$ cd web
$ npm install
$ npm run dev
```
open http://localhost:8080/ in browser
